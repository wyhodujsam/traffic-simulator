## Task 1.1.2: Create TrafficSimulatorApplication.java

<read_first>
- /home/sebastian/traffic-simulator/.planning/phases/01-project-bootstrap-infrastructure/01-RESEARCH.md (section 5)
</read_first>

<action>
Create `backend/src/main/java/com/trafficsimulator/TrafficSimulatorApplication.java`:

```java
package com.trafficsimulator;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.scheduling.annotation.EnableScheduling;

@SpringBootApplication
@EnableScheduling
public class TrafficSimulatorApplication {
    public static void main(String[] args) {
        SpringApplication.run(TrafficSimulatorApplication.class, args);
    }
}
```

The `@EnableScheduling` annotation is required for the `@Scheduled` tick emitter in Plan 1.4.
</action>
