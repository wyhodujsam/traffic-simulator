# Plan 09-02: Predefined Scenario Map Files

## Goal

Create 3 new predefined map JSON files that exercise different traffic phenomena: phantom traffic jams on a long corridor, merge congestion from an on-ramp, and lane-blocking from a construction zone. Each must pass MapValidator and load correctly via MapLoader.

## Tasks

### Task 1: phantom-jam-corridor.json

<read_first>
- `backend/src/main/resources/maps/straight-road.json` (format reference)
- `backend/src/main/java/com/trafficsimulator/config/MapValidator.java` (validation rules)
- `backend/src/main/java/com/trafficsimulator/config/MapConfig.java` (schema)
</read_first>

<action>
Create `backend/src/main/resources/maps/phantom-jam-corridor.json`:

**Purpose:** Showcase phantom traffic jam emergence on an unobstructed road with high density.

**Layout:**
- Long 2000m, 2-lane corridor
- High defaultSpawnRate (3.0 veh/s) to create density
- Moderate speedLimit (25 m/s = 90 km/h) — realistic highway speed

```json
{
  "id": "phantom-jam-corridor",
  "name": "Phantom Jam Corridor",
  "description": "Long 2-lane highway with high traffic density. Watch spontaneous stop-and-go waves emerge from pure car-following physics — no obstacles needed.",
  "nodes": [
    { "id": "entry", "type": "ENTRY", "x": 50.0, "y": 200.0 },
    { "id": "exit",  "type": "EXIT",  "x": 2050.0, "y": 200.0 }
  ],
  "roads": [
    {
      "id": "corridor",
      "name": "Highway Corridor",
      "fromNodeId": "entry",
      "toNodeId": "exit",
      "length": 2000.0,
      "speedLimit": 25.0,
      "laneCount": 2
    }
  ],
  "intersections": [],
  "spawnPoints": [
    { "roadId": "corridor", "laneIndex": 0, "position": 0.0 },
    { "roadId": "corridor", "laneIndex": 1, "position": 0.0 }
  ],
  "despawnPoints": [
    { "roadId": "corridor", "laneIndex": 0, "position": 2000.0 },
    { "roadId": "corridor", "laneIndex": 1, "position": 2000.0 }
  ],
  "defaultSpawnRate": 3.0
}
```
</action>

<acceptance_criteria>
- JSON is valid and passes MapValidator (no errors)
- MapLoader builds a RoadNetwork with 1 road, 2 lanes, 2 spawn points, 2 despawn points
- defaultSpawnRate is 3.0
- Canvas will be wide (~2100px) — acceptable for scrollable view
</acceptance_criteria>

<action>
Create `backend/src/main/resources/maps/highway-merge.json`:

**Purpose:** Demonstrate merge congestion where an on-ramp joins a highway.

**Layout:**
- Main highway: 2-lane, 1000m total, split into two segments (before merge, after merge)
- On-ramp: 1-lane, 200m, joins at merge point (300m mark)
- After merge point: 2 lanes continue to exit

```json

  "name": "Highway Merge",
  "description": "2-lane highway with a 1-lane on-ramp merging at the 300m mark. Observe how merge points create bottleneck congestion.",
  "nodes": [
    { "id": "main_entry",  "type": "ENTRY",        "x": 50.0,   "y": 200.0 },
    { "id": "ramp_entry",  "type": "ENTRY",        "x": 200.0,  "y": 50.0 },
    { "id": "merge_point", "type": "INTERSECTION", "x": 350.0,  "y": 200.0 },
    { "id": "main_exit",   "type": "EXIT",         "x": 1050.0, "y": 200.0 }
  ],
  "roads": [
    {
      "id": "main_before",
      "name": "Highway Before Merge",
      "fromNodeId": "main_entry",
      "toNodeId": "merge_point",
      "length": 300.0,
      "speedLimit": 30.0,
      "laneCount": 2
    },
    {
      "id": "ramp",
      "name": "On-Ramp",
      "fromNodeId": "ramp_entry",
      "toNodeId": "merge_point",
      "length": 200.0,
      "speedLimit": 20.0,
      "laneCount": 1
    },
    {
      "id": "main_after",
      "name": "Highway After Merge",
      "fromNodeId": "merge_point",
      "toNodeId": "main_exit",
      "length": 700.0,
      "speedLimit": 30.0,
      "laneCount": 2
    }
  ],
  "intersections": [
    {
      "nodeId": "merge_point",
      "type": "PRIORITY",
      "signalPhases": null
    }
  ],
  "spawnPoints": [
    { "roadId": "main_before", "laneIndex": 0, "position": 0.0 },
    { "roadId": "main_before", "laneIndex": 1, "position": 0.0 },
    { "roadId": "ramp",        "laneIndex": 0, "position": 0.0 }
  ],
  "despawnPoints": [
    { "roadId": "main_after", "laneIndex": 0, "position": 700.0 },
    { "roadId": "main_after", "laneIndex": 1, "position": 700.0 }
  ],
  "defaultSpawnRate": 2.0
}
```
</action>

<acceptance_criteria>
- JSON passes MapValidator — all node references valid, lane counts 1-2, positive lengths/speeds
- MapLoader builds RoadNetwork with 3 roads, 1 PRIORITY intersection at merge_point
- Intersection has inbound roads: main_before, ramp; outbound: main_after
- defaultSpawnRate is 2.0
</acceptance_criteria>

```json
  ],
  "roads": [
    {
      "id": "main_road",
      "name": "Highway",
      "fromNodeId": "entry",
      "toNodeId": "exit",
      "length": 600.0,
      "speedLimit": 27.8,
      "laneCount": 3
    }
  ],
  "intersections": [],
  "spawnPoints": [
    { "roadId": "main_road", "laneIndex": 0, "position": 0.0 },
    { "roadId": "main_road", "laneIndex": 1, "position": 0.0 },
    { "roadId": "main_road", "laneIndex": 2, "position": 0.0 }
  ],
  "despawnPoints": [
    { "roadId": "main_road", "laneIndex": 0, "position": 600.0 },
    { "roadId": "main_road", "laneIndex": 1, "position": 600.0 },
    { "roadId": "main_road", "laneIndex": 2, "position": 600.0 }
  ],

}
```
</action>

<acceptance_criteria>

- speedLimit 27.8 m/s (100 km/h)
