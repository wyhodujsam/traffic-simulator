### Task 4: Enhance GET /api/maps to return MapInfoDto list

<read_first>
- `backend/src/main/java/com/trafficsimulator/controller/SimulationController.java` (listMaps method)
- `backend/src/main/java/com/trafficsimulator/config/MapLoader.java` (loadFromClasspath)
</read_first>

<action>
Modify `SimulationController.listMaps()`:
1. Change return type from `List<String>` to `List<MapInfoDto>`
2. For each map JSON found, parse it with ObjectMapper to read id, name, description
3. Build `MapInfoDto` for each and return the list
4. Inject `ObjectMapper` into the controller (add to constructor)

```java
@GetMapping("/maps")
public List<MapInfoDto> listMaps() throws IOException {
    PathMatchingResourcePatternResolver resolver = new PathMatchingResourcePatternResolver();
    Resource[] resources = resolver.getResources("classpath:maps/*.json");
    List<MapInfoDto> maps = new ArrayList<>();
    for (Resource resource : resources) {
        try (InputStream is = resource.getInputStream()) {
            MapConfig config = objectMapper.readValue(is, MapConfig.class);
            maps.add(MapInfoDto.builder()
                .id(config.getId())
                .name(config.getName())
                .description(config.getDescription())
                .build());
        }
    }
    return maps;
}
```

Inject ObjectMapper: add `private final ObjectMapper objectMapper;` field (already available as Spring bean).
</action>

</acceptance_criteria>

### Task 6: Fix handleLoadMap in CommandDispatcher

<read_first>
- `backend/src/main/java/com/trafficsimulator/engine/CommandDispatcher.java` (handleLoadMap, handleStop)
- `backend/src/main/java/com/trafficsimulator/engine/SimulationEngine.java` (clearAllVehicles, setRoadNetwork)
</read_first>

<action>
Fix `CommandDispatcher.handleLoadMap()` to address all gaps:

1. **Call clearAllVehicles()** — add `engine.clearAllVehicles()` after setting STOPPED status
2. **Reset obstacles** — add `if (obstacleManager != null) obstacleManager.clearAll(roadNetwork)` (clear obstacles from old network before loading new one). If `ObstacleManager` lacks `clearAll`, add it: iterate all lanes in all roads, clear obstacle lists.
3. **Wire defaultSpawnRate** — after loading new network, read `config.getDefaultSpawnRate()` and apply: `engine.setSpawnRate(rate)` and `vehicleSpawner.setVehiclesPerSecond(rate)`. This requires MapLoader to expose the loaded config's defaultSpawnRate. Simplest: have MapLoader return a wrapper or store the last config. Alternative: have handleLoadMap parse the map config separately just for the rate — but cleaner to extend MapLoader with a `getLastLoadedConfig()` or return a `LoadedMap` record containing both RoadNetwork and defaultSpawnRate.

Recommended approach — create a simple record:
```java
// In MapLoader.java, add:
public record LoadedMap(RoadNetwork network, double defaultSpawnRate) {}

// Change loadFromClasspath return type to LoadedMap
public LoadedMap loadFromClasspath(String resourcePath) throws IOException {
    // ... existing code ...
    RoadNetwork network = buildRoadNetwork(config);
    return new LoadedMap(network, config.getDefaultSpawnRate());
}
```

Then in handleLoadMap:
```java
private void handleLoadMap(SimulationCommand.LoadMap cmd) {
    if (mapLoader == null) {
        log.warn("MapLoader not available — ignoring LoadMap command");
        return;
    }
    // Stop and clear
    engine.setStatus(SimulationStatus.STOPPED);
    engine.getTickCounter().set(0);
    engine.clearAllVehicles();
    if (obstacleManager != null) {
        RoadNetwork oldNetwork = engine.getRoadNetwork();
        if (oldNetwork != null) {
            obstacleManager.clearAll(oldNetwork);
        }
    }
    if (vehicleSpawner != null) {
        vehicleSpawner.reset();
    }

    try {
        MapLoader.LoadedMap loaded = mapLoader.loadFromClasspath("maps/" + cmd.mapId() + ".json");
        engine.setRoadNetwork(loaded.network());
        // Apply default spawn rate from map config
        engine.setSpawnRate(loaded.defaultSpawnRate());
        if (vehicleSpawner != null) {
            vehicleSpawner.setVehiclesPerSecond(loaded.defaultSpawnRate());
        }
        engine.setLastError(null); // clear any previous error
        log.info("Map loaded: {} (spawn rate: {} veh/s)", cmd.mapId(), loaded.defaultSpawnRate());
    } catch (Exception e) {
        log.error("Failed to load map {}: {}", cmd.mapId(), e.getMessage());
        engine.setLastError("Failed to load map '" + cmd.mapId() + "': " + e.getMessage());
    }
}
```
</action>

<acceptance_criteria>
- LoadMap clears all vehicles from old network before loading new one
- LoadMap clears obstacles from old network
- LoadMap applies defaultSpawnRate from map config to engine and spawner
- LoadMap error is stored in engine.lastError for frontend display
- MapLoader returns LoadedMap record with both network and defaultSpawnRate
</acceptance_criteria>
