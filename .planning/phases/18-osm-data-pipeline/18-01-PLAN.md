       **Package-private method: MapConfig convertOsmToMapConfig(String json, BboxRequest bbox)**
       This is the core converter, tested independently:

       a. Parse JSON with objectMapper.readTree(json). Extract "elements" array.
       b. Pass 1: Collect nodes into Map<Long, OsmNode>, ways into List<OsmWay>. Parse tags from each element.
       c. Pass 2: Count node references across all ways → Map<Long, Integer> nodeRefCount
       d. Identify intersection nodes: refCount >= 2 OR tagged highway=traffic_signals
       e. Identify terminal nodes: first/last node of a way with refCount == 1
       f. Identify roundabout ways: tagged junction=roundabout

       **For each OsmWay, generate RoadConfig(s):**
       - Skip ways with fewer than 2 resolvable node coordinates
       - Road ID: "osm-{wayId}-fwd" (and "osm-{wayId}-rev" for bidirectional)
       - fromNodeId/toNodeId: "osm-{firstNodeId}" / "osm-{lastNodeId}" (or reversed for -rev)
       - length: sum of Haversine distances between consecutive nodes (in meters). Skip if < 1.0m.
       - speedLimit: from highway type defaults table (motorway=36.1, trunk=27.8, primary=19.4, secondary=16.7, tertiary=13.9, unclassified=11.1, residential=8.3, living_street=2.8 m/s)
       - laneCount: from tags.get("lanes") if present, parsed to int, clamped to [1,4]. Else use highway-type default (motorway/trunk/primary/secondary=2, others=1)
       - oneway handling:
         - oneway=yes OR oneway=true OR junction=roundabout → generate only -fwd road
         - oneway=-1 → generate only -fwd road BUT reverse node order first
         - default (no oneway tag) → generate both -fwd and -rev roads

       **Generate NodeConfigs:**
       - For each unique node referenced as fromNodeId or toNodeId of any generated road:
         - id: "osm-{nodeId}"
         - type: "INTERSECTION" if in intersection set, "ENTRY" if terminal and is a from-node, "EXIT" if terminal and is a to-node
         - x: lonToX(lon, bbox.west(), bbox.east()) using linear projection with CANVAS_W=1600, CANVAS_PADDING=50
         - y: latToY(lat, bbox.south(), bbox.north()) using linear projection with CANVAS_H=1200, CANVAS_PADDING=50, inverted Y axis

       **Generate IntersectionConfigs:**
       - For each intersection node:
         - If node has tag highway=traffic_signals → type "SIGNAL", generate default 2-phase signal (30s each, alternating inbound road groups)
         - If node is part of a junction=roundabout way → type "ROUNDABOUT", roundaboutCapacity=8
         - Else → type "PRIORITY"

       **Generate SpawnPointConfigs / DespawnPointConfigs:**
       - For each terminal node:
         - Find all roads where this node is fromNodeId → add SpawnPointConfig for each lane (position=0.0)
         - Find all roads where this node is toNodeId → add DespawnPointConfig for each lane (position=road.length)

       **Assemble MapConfig:**
       - id: "osm-bbox-{south}-{west}-{north}-{east}" (truncated to 4 decimals)
       - name: "OSM Import"
       - defaultSpawnRate: 1.0
       - If roads list is empty after filtering, throw IllegalStateException("No roads found in selected area")

       **Haversine helper (private static):**
       ```java
       private static double haversineMeters(double lat1, double lon1, double lat2, double lon2) {
           final double R = 6371000;
           double dLat = Math.toRadians(lat2 - lat1);
           double dLon = Math.toRadians(lon2 - lon1);
           double a = Math.sin(dLat/2)*Math.sin(dLat/2)
                    + Math.cos(Math.toRadians(lat1))*Math.cos(Math.toRadians(lat2))
                    * Math.sin(dLon/2)*Math.sin(dLon/2);
           return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
       }
       ```

       **Coordinate projection helpers (private static, constants CANVAS_W=1600, CANVAS_H=1200, CANVAS_PADDING=50):**
       ```java
       private static double lonToX(double lon, double west, double east) {
           double usable = CANVAS_W - 2 * CANVAS_PADDING;
           return CANVAS_PADDING + (lon - west) / (east - west) * usable;
       }
       private static double latToY(double lat, double south, double north) {
           double usable = CANVAS_H - 2 * CANVAS_PADDING;
           return CANVAS_PADDING + (north - lat) / (north - south) * usable;
       }
       ```
