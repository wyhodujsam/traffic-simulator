5. Define interface for bbox bounds callback:
   ```typescript
   interface BboxInfo {
     readonly south: number;
     readonly west: number;
     readonly north: number;
     readonly east: number;
     readonly widthMeters: number;
     readonly heightMeters: number;
   }

   interface BoundingBoxMapProps {
     readonly onBoundsChange?: (bbox: BboxInfo) => void;
     readonly center?: [number, number];
     readonly zoom?: number;
     readonly onViewChange?: (center: [number, number], zoom: number) => void;
   }
   ```

6. Create `BoundingBoxLayer` inner component (uses `useMap()` hook):
   - Create a ref for L.Rectangle
   - On mount and on `moveend`/`zoomend` events:
     - Get map bounds via `map.getBounds()`
     - Calculate bbox as padded inner rectangle: inset by 20% from each edge of the visible area
       ```typescript
       const bounds = map.getBounds();
       const latPad = (bounds.getNorth() - bounds.getSouth()) * 0.2;
       const lngPad = (bounds.getEast() - bounds.getWest()) * 0.2;
       const bboxBounds = L.latLngBounds(
         [bounds.getSouth() + latPad, bounds.getWest() + lngPad],
         [bounds.getNorth() - latPad, bounds.getEast() - lngPad]
       );
       ```
     - If rectRef.current exists, call `setBounds(bboxBounds)` — else create new `L.rectangle(bboxBounds, { color: '#0088ff', weight: 2, fillOpacity: 0.1, interactive: false })` and add to map
     - Calculate width/height in meters using `map.distance()`:
       ```typescript
       const sw = bboxBounds.getSouthWest();
       const ne = bboxBounds.getNorthEast();
       const widthMeters = map.distance(sw, L.latLng(sw.lat, ne.lng));
       const heightMeters = map.distance(sw, L.latLng(ne.lat, sw.lng));
       ```
     - Call `onBoundsChange` callback with `{ south, west, north, east, widthMeters, heightMeters }`
   - Also call `onViewChange` with current center and zoom on moveend/zoomend (for persisting position)
   - Cleanup: remove rectangle and event listeners on unmount
   - Return null (this is a side-effect-only component)

7. Export `BoundingBoxMap` component:
   - Default center: `[52.2297, 21.0122]` (Warsaw, per CONTEXT.md and RESEARCH.md)
   - Default zoom: 14
   - Use props `center` and `zoom` if provided (for restoring saved position)
   - Render `<MapContainer>` with `style={{ height: '100%', width: '100%' }}`
   - Inside: `<TileLayer>` with OpenStreetMap URL and attribution
   - Inside: `<BoundingBoxLayer onBoundsChange={onBoundsChange} onViewChange={onViewChange} />`

CRITICAL per RESEARCH.md pitfalls:
- Use react-leaflet v4 API (useMap hook, NOT withLeaflet HOC)
- MapContainer MUST have explicit height (100% with parent having explicit height)
- Import 'leaflet/dist/leaflet.css' in this file
