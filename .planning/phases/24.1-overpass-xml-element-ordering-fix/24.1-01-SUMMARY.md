## Decisions Made

- **Single-pass union recursion** chosen over alternative two-pass forms (e.g. `out meta; >; out;`). Rationale documented in `24.1-CONTEXT.md` Decisions table — single-pass produces standard nodes-before-ways XML and is what most OSM tooling expects, including osm2streets-cli (verified in CONTEXT under "Bug summary" — direct curl against Overpass with this form returns 161 roads / 132 intersections for the Modlin bbox).
- **`out body qt;` instead of `out skel qt;`** — `body` keeps full node tags (e.g. `highway=traffic_signals`) so downstream osm2streets can attach signal metadata. `skel` would drop those tags. Phase 24 lane-level rendering depends on this.
- **3 unit tests, not 2** — added the SWNE bbox-order test as a co-validation that the GREEN edit didn't accidentally swap the `bbox.south(), bbox.west(), bbox.north(), bbox.east()` argument order. Cheap insurance.
