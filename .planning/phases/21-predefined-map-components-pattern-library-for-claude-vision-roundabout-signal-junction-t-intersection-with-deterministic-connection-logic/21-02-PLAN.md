Algorithm per RESEARCH §"Connection / Stitching Algorithm — Decision":

```
Input:  ctx (already populated by each component.expand), connections List<Connection>
1. armIndex: Map<String armKey "rb1.north", ArmRecord{entryNodeId, exitNodeId, Point2D worldPos}>
   — built from each ComponentSpec.armEndpoints() + naming convention prefix + "n_<arm>"/"n_<arm>_exit"
2. For each Connection(a,b):
     posA = armIndex.get(a.componentId+"."+a.armName).worldPos
     posB = armIndex.get(b.componentId+"."+b.armName).worldPos
     if dist(posA,posB) <= MERGE_TOLERANCE_PX (5.0):
         mergedId = "merged_" + a.componentId + "_" + a.armName + "_" + b.componentId + "_" + b.armName
         // Create new INTERSECTION node at midpoint
         ctx.nodes.add(new NodeConfig(mergedId, INTERSECTION, midX, midY))
         // Rewrite all roads whose from/to equals any of the 4 deprecated arm endpoints
         Set<String> deadIds = { entryA, exitA, entryB, exitB }
         for road r in ctx.roads: if r.from in deadIds → r.from = mergedId ; if r.to in deadIds → r.to = mergedId
         // Drop the 4 deprecated nodes
         ctx.nodes.removeIf(n -> deadIds.contains(n.id))
         // Drop spawn/despawn whose roadId referenced a deleted ENTRY/EXIT
         ctx.spawnPoints.removeIf(sp -> the road's from-node is no longer ENTRY)
         ctx.despawnPoints.removeIf(dp -> the road's to-node is no longer EXIT)
         // Record mergedId as a PRIORITY intersection (default control)
         //   Exception: if either side was a ROUNDABOUT ring arm endpoint, the ring's intersection
         //   already owns the ring node, so fused arm nodes inherit ROUNDABOUT control —
         //   in practice only happens when two roundabouts kiss; for MVP, always add PRIORITY.
         ctx.intersections.add(new IntersectionConfig(mergedId, PRIORITY, 24))
     else:
         throw new ExpansionException("Arms " + a + " and " + b +
             " are " + dist + "px apart; insert a STRAIGHT_SEGMENT between them.");
3. After processing all connections: drop orphaned intersection nodes whose nodeId
   appears in no road (defensive — RESEARCH Risks #3).
4. mapValidator.validate(ctx.toMapConfig(...)) MUST return [].
```

Constants:
- `MERGE_TOLERANCE_PX = 5.0` (per RESEARCH Pattern 3)
- `MIN_NODE_SEPARATION_PX = 5.0` used for post-expansion sanity (RESEARCH Pitfall 3)

       ```java
       public MapConfig expand(List<ComponentSpec> components, List<Connection> connections) {
           for (ComponentSpec c : components) validateId(c.id());
           ExpansionContext ctx = new ExpansionContext();
           for (ComponentSpec c : components) c.expand(ctx);

           for (Connection conn : connections) {
               ArmRecord ra = ctx.lookupArm(conn.a());
               ArmRecord rb = ctx.lookupArm(conn.b());
               double d = ra.worldPos().distance(rb.worldPos());
               if (d > MERGE_TOLERANCE_PX) {
                   throw new ExpansionException(String.format(
                       "Arms %s.%s and %s.%s are %.1fpx apart (>%.1f). Insert a STRAIGHT_SEGMENT between them.",
                       conn.a().componentId(), conn.a().armName(),
                       conn.b().componentId(), conn.b().armName(), d, MERGE_TOLERANCE_PX));
               }
               // Merge
               String mergedId = String.format("merged__%s_%s__%s_%s",
                   conn.a().componentId(), conn.a().armName(),
                   conn.b().componentId(), conn.b().armName());
               double mx = (ra.worldPos().x + rb.worldPos().x) / 2.0;
               double my = (ra.worldPos().y + rb.worldPos().y) / 2.0;
               ctx.nodes.add(new NodeConfig(mergedId, NodeType.INTERSECTION, mx, my));
               Set<String> dead = Set.of(ra.entryNodeId(), ra.exitNodeId(), rb.entryNodeId(), rb.exitNodeId());
               ctx.rewriteRoadEndpoint(dead, mergedId);
               ctx.dropSpawnDespawnForRoadsReferencing(dead);
               ctx.dropNodes(dead);
               ctx.intersections.add(IntersectionConfig.priority(mergedId, 24));
           }

           // Drop intersection configs whose nodeId no longer appears in any road (defensive).
           Set<String> referencedNodes = ctx.roads.stream()
               .flatMap(r -> Stream.of(r.getFromNodeId(), r.getToNodeId()))
               .collect(Collectors.toSet());
           ctx.intersections.removeIf(ic -> !referencedNodes.contains(ic.getNodeId()));

           MapConfig cfg = ctx.toMapConfig("vision-generated", "AI Generated Map (component library)");
           List<String> errors = mapValidator.validate(cfg);
           if (!errors.isEmpty()) {
               throw new ExpansionException("Stitched MapConfig failed validation: " + String.join(", ", errors));
           }
           return cfg;
       }
       ```
