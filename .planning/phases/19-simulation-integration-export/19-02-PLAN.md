Add a new endpoint to `SimulationController.java`:

```java
@PostMapping("/command/load-config")
public Map<String, String> loadConfig(@RequestBody MapConfig config) {
    simulationEngine.enqueue(new SimulationCommand.LoadConfig(config));
    return Map.of("status", "ok");
}
```

This is a dedicated endpoint because `LoadConfig` requires a full `MapConfig` JSON body (not the simple `CommandDto` with string type used by `/api/command`). The existing `CommandDispatcher.handleLoadConfig()` already handles the command correctly — it stops the simulation, clears vehicles, loads the config via `MapLoader.loadFromConfig()`, and sets up the road network.

Do NOT modify the existing `POST /api/command` endpoint or `CommandDto`. The dedicated endpoint is cleaner and avoids overloading the generic command DTO.

   ```typescript
   const handleRunSimulation = useCallback(async () => {
     if (!mapConfig) return;
     setSimulationLoading(true);
     try {
       const response = await fetch('/api/command/load-config', {
         method: 'POST',
         headers: { 'Content-Type': 'application/json' },
         body: JSON.stringify(mapConfig),
       });
       if (!response.ok) {
         const err = await response.json().catch(() => ({ error: 'Unknown error' }));
         throw new Error((err as { error?: string }).error ?? `HTTP ${response.status}`);
       }
       // Send START command via STOMP to begin simulation
       const sendCommand = useSimulationStore.getState().sendCommand;
       if (sendCommand) {
         sendCommand({ type: 'START' });
       }
       // Navigate to simulation page
       navigate('/');
     } catch (e: unknown) {
       const message = e instanceof Error ? e.message : 'Failed to load config';
       setFetchError(message);
       setSidebarState('error');
     } finally {
       setSimulationLoading(false);
     }
   }, [mapConfig, navigate]);
   ```
5. Pass `onRunSimulation={handleRunSimulation}` and `simulationLoading={simulationLoading}` to MapSidebar.
6. Import `useSimulationStore` at top of file.
