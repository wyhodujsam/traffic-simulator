Create `RoadGraphPreview.tsx` that renders road graph nodes and edges as a Leaflet SVG overlay on top of the existing map.

The component receives the `mapConfig` object (the raw JSON returned by `/api/osm/fetch-roads`). The MapConfig structure has:
- `nodes`: array of `{ id, x, y }` where x/y are pixel coordinates. The converter uses Mercator projection, so the nodes need to be converted back to lat/lng using the bbox bounds for display on Leaflet.
- `roads`: array of `{ id, fromNodeId, toNodeId, laneCount, ... }` — draw a line from fromNode to toNode.

**Approach — use react-leaflet's built-in components:**
1. Use `CircleMarker` from react-leaflet for nodes (small dots, radius 4px, color #4af).
2. Use `Polyline` from react-leaflet for road edges (lines between nodes, color #4af with opacity 0.7, weight proportional to laneCount: `Math.max(2, laneCount * 1.5)`).
3. The component needs a coordinate mapping function. The mapConfig nodes have x/y in pixel space. To convert back to lat/lng:
   - Read the bbox from the MapPage (pass as prop).
   - Linear interpolation: `lat = south + (y / maxY) * (north - south)`, `lng = west + (x / maxX) * (east - west)`. Find maxX/maxY from the node coordinates.
4. Wrap all markers and polylines in a React fragment — react-leaflet components render directly inside MapContainer.

- Since BoundingBoxMap wraps MapContainer, either: (a) pass RoadGraphPreview as children to BoundingBoxMap, or (b) render it alongside using a shared map ref. Preferred approach: modify BoundingBoxMap to accept `children` prop and render them inside MapContainer.

Type the mapConfig prop loosely (the raw JSON shape) — define a local interface:

In `MapPage.tsx`:
1. Create a `handleExportJson` callback:
   ```typescript
   const handleExportJson = useCallback(() => {
     if (!mapConfig) return;
     const json = JSON.stringify(mapConfig, null, 2);
     const blob = new Blob([json], { type: 'application/json' });
     const url = URL.createObjectURL(blob);
     const a = document.createElement('a');
     a.href = url;
     a.download = 'map-config.json';
     a.click();
     URL.revokeObjectURL(url);
   }, [mapConfig]);
   ```
2. Pass `onExportJson={handleExportJson}` to `MapSidebar`.
