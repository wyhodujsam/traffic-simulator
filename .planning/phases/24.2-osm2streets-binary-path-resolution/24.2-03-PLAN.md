<objective>
Remove the `-Dosm2streets.binary-path=bin/osm2streets-cli-linux-x64` override from `frontend/playwright.config.ts`'s backend webServer command. Plan 01's smart resolver in `Osm2StreetsConfig.getBinaryPath()` makes the override unnecessary: when Playwright spawns `cd ../backend && mvn spring-boot:run` (cwd=`backend/`), the resolver walks up to find the project root (`.git` + root `pom.xml`) and resolves `backend/bin/osm2streets-cli-linux-x64` against it, producing the correct absolute path automatically.

Preserve the `-Dosm.overpass.urls=http://localhost:18086` override — that one is Phase 24.1 Plan 03's contract for redirecting Overpass calls to the local fixture server (`overpass-fixture-server.cjs`), unrelated to the binary path.

Purpose: complete the override removal across the test surface. After Plan 02 dropped the backend integration test override and this plan drops the e2e override, NO test in the project pre-supplies `osm2streets.binary-path` — the production default is what every test exercises.
Output: One file modified (`frontend/playwright.config.ts`), webServer command shortened from two `-D` properties to one, with an updated comment block.
</objective>
