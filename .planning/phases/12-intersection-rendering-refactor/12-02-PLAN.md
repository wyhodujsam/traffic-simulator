# Plan 12.2: Backend — IntersectionDto endpoint for frontend rendering

The frontend needs to know where intersections are and how big they are so it can draw them as filled boxes. Currently only traffic lights carry position data, but that is per-approach-road, not the intersection itself.

```java
package com.trafficsimulator.dto;

import lombok.AllArgsConstructor;
import lombok.Builder;
import lombok.Data;
import lombok.NoArgsConstructor;

@Data
@Builder
@NoArgsConstructor
@AllArgsConstructor
public class IntersectionDto {
    private String id;
    private double x;          // center X (from node coordinates)
    private double y;          // center Y (from node coordinates)
    private double size;       // total box size in pixels
}
```

2. Add `GET /api/intersections` endpoint in `SimulationController.java`:
```java
@GetMapping("/intersections")
public List<IntersectionDto> getIntersections() {
    RoadNetwork network = simulationEngine.getRoadNetwork();
    if (network == null) return List.of();

    List<IntersectionDto> result = new ArrayList<>();
    for (Intersection ixtn : network.getIntersections().values()) {
        // Get node coordinates — intersection center is the node position
        // Node coords are available through connected roads
        double cx = 0, cy = 0;
        boolean found = false;
        for (String roadId : ixtn.getConnectedRoadIds()) {
            Road road = network.getRoads().get(roadId);
            if (road == null) continue;
            if (road.getToNodeId().equals(ixtn.getId())) {
                cx = road.getEndX(); cy = road.getEndY(); found = true; break;
            }
            if (road.getFromNodeId().equals(ixtn.getId())) {
                cx = road.getStartX(); cy = road.getStartY(); found = true; break;
            }
        }
        if (!found) continue;

        result.add(IntersectionDto.builder()
            .id(ixtn.getId())
            .x(cx).y(cy)
            .size(ixtn.getIntersectionSize())
            .build());
    }
    return result;
}
```
