## Task 12.1.3: Compute clip distances in SimulationController /api/roads

<read_first>
- /home/sebastian/traffic-simulator/backend/src/main/java/com/trafficsimulator/controller/SimulationController.java
- /home/sebastian/traffic-simulator/backend/src/main/java/com/trafficsimulator/config/MapLoader.java
- /home/sebastian/traffic-simulator/backend/src/main/java/com/trafficsimulator/model/RoadNetwork.java
- /home/sebastian/traffic-simulator/backend/src/main/java/com/trafficsimulator/model/Intersection.java
</read_first>

<action>
1. In the `getRoads()` method of `SimulationController`, after building each `RoadDto`, compute clip values:

For each road:
- If road's `fromNodeId` matches an intersection with `intersectionSize > 0`, set `clipStart = intersectionSize / 2` (half the box extends along the road).
- If road's `toNodeId` matches an intersection with `intersectionSize > 0`, set `clipEnd = intersectionSize / 2`.
- Default both to 0 if no intersection on that end.

This requires access to the `MapConfig` intersection data. Options:
- Store the `intersectionSize` on the `Intersection` model (cleanest) during `MapLoader.buildRoadNetwork()`.
- Or look it up from a stored `MapConfig` reference.

Preferred: Add `intersectionSize` field to `Intersection.java` model, populate it in `MapLoader`, then use it in `SimulationController.getRoads()`.

2. In `MapLoader.buildRoadNetwork()`, when building `Intersection` objects, set:
```java
.intersectionSize(ic.getIntersectionSize())
```

3. In `Intersection.java` model, add:
```java
@Builder.Default
private double intersectionSize = 0;
```

4. In `SimulationController.getRoads()`, after building the road DTO, compute clips:
```java
// Compute clip distances based on intersection sizes
Map<String, Intersection> intersections = network.getIntersections();
for (RoadDto dto : result) {
    // find matching road model to get node IDs
}
```

Actually simpler: compute during the existing loop where we have access to the `Road` model:
```java
Road road = ...;  // already in the loop
double clipStart = 0, clipEnd = 0;
Intersection fromIxtn = network.getIntersections().get(road.getFromNodeId());
Intersection toIxtn = network.getIntersections().get(road.getToNodeId());
if (fromIxtn != null && fromIxtn.getIntersectionSize() > 0) {
    clipStart = fromIxtn.getIntersectionSize() / 2.0;
}
if (toIxtn != null && toIxtn.getIntersectionSize() > 0) {
    clipEnd = toIxtn.getIntersectionSize() / 2.0;
}
// ... .clipStart(clipStart).clipEnd(clipEnd) in builder
```
</action>
